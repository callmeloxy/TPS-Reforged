# TPS Reforged

Mod de surveillance des TPS et MSPT Minecraft avec affichage global et par dimension, couleurs configurables, format personnalisable et support de l’actionbar.

## Présentation

TPS Reforged est un mod utilitaire léger conçu pour rendre les performances d’un serveur Minecraft plus faciles à surveiller, avec les TPS/MSPT globaux, les informations de tick par dimension, des options d’affichage configurables et un affichage actionbar optionnel.

## Fonctionnalités

- Affichage des TPS et MSPT globaux du serveur
- Affichage des TPS/MSPT par dimension
- Format compact en une ligne par dimension
- Couleurs configurables
- Seuils TPS/MSPT configurables
- Noms d’affichage des dimensions configurables
- Ajout automatique des dimensions inconnues dans la configuration
- Ordre d’affichage des dimensions personnalisable
- Possibilité de masquer certaines dimensions
- Format de sortie configurable
- Affichage TPS optionnel dans l’actionbar
- Support de Minecraft `1.21.1`
- Développement ciblant `Fabric`
